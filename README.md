# GridSense-Agent
 面向智能电网巡检的多模态感知与长链推理系统
项目名称： "GridSense-Agent: 面向智能电网巡检的多模态感知与长链推理系统"

项目解决的核心痛点：
传统的电力巡检机器人依赖预设脚本，在面对复杂非结构化环境（如变电站极端天气、动态障碍物）时，视觉感知深度精度不足，且缺乏对电力缺陷（如绝缘子破损、异物搭挂）的逻辑推理能力。本项目旨在解决巡检过程中“感知精度低”与“决策不智能”的痛点，将高精度双目深度感知与大模型推理闭环。

核心逻辑流（长链推理 + 多 Agent 协作）：
本项目构建了一个三阶段协作的 Agent 架构：

感知 Agent (Perception Agent)： 核心算法集成我研究的 MMGA-KAN Net（多模态图注意力 Kolmogorov-Arnold 网络），实现亚像素级的双目立体匹配，为系统提供实时的、具备物理约束的高精度 3D 环境地图。

分析推理 Agent (Analytical Reasoning Agent)： 采用长链推理（Chain-of-Thought）。当感知 Agent 发现潜在异常时，该 Agent 会结合多模态信息（图像+红外温度+深度特征）进行逻辑推演，例如：“检测到绝缘子颜色异常 -> 结合红外数据发现局部高温 -> 推理结论：存在闪络风险，建议近距离复检”。

决策规划 Agent (Task Planning Agent)： 基于推理结论，自主调用路径规划工具，实现巡检路线的实时重调度，完成从“发现异常”到“闭环处置”的全流程自主化。
